# 🍅 Pomodoro To-Do

Um aplicativo de produtividade que reúne duas das ferramentas mais utilizadas para organização pessoal: o **Método Pomodoro** e uma **Lista de Tarefas**.

O objetivo do projeto é oferecer um ambiente simples e intuitivo para organizar atividades e manter o foco durante os períodos de estudo ou trabalho.

## ✨ Funcionalidades

### 🍅 Pomodoro

* Cronômetro Pomodoro
* Pausa e continuação do cronômetro
* Reinício do ciclo
* Alteração da duração dos ciclos
* Modos:
  * Pomodoro
  * Descanso Curto
  * Descanso Longo
* Sons ao iniciar e pausar o cronômetro

### 📝 Lista de tarefas

* Criar tarefas
* Editar tarefas
* Excluir tarefas
* Marcar tarefas como concluídas

### 💾 Persistência de dados

* Armazenamento automático utilizando LocalStorage

### 📱 Interface

* Layout responsivo
* Interface moderna e intuitiva

## ⚙️ Como executar o projeto

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Entre na pasta do projeto:

```bash
cd <nome-do-projeto>
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Para gerar a versão de produção:

```bash
npm run build
```
